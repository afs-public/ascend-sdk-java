# DowJonesDate

Custom date object to present related Dow Jones dates because any value can be null, Month is represented randomly like June, Jun, 6 etc. etc.


## Fields

| Field               | Type                | Required            | Description         | Example             |
| ------------------- | ------------------- | ------------------- | ------------------- | ------------------- |
| `day`               | *Optional\<String>* | :heavy_minus_sign:  | Day                 | 1                   |
| `month`             | *Optional\<String>* | :heavy_minus_sign:  | Month               | June                |
| `year`              | *Optional\<String>* | :heavy_minus_sign:  | Year                | 2020                |
